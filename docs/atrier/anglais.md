#### 1.1.1 La conjugaison

Les verbes régulier ont la même conjugaison pour tous les sujets à l'exception de la 3eme personne du singulier ou on devra rajouter un "**S**" à la fin.

Rappel sur les sujets

|Francais | Anglais|
|:-------:|:------:|
|Je       | I|
|Tu/Vous  | You|
|<span style="color:red">Il/Elle</span>  | <span style="color:red">He/She/It</span>|
|Nous     | You|
|Vous     | We|
|Ils/Elles| They|

En anglais les genres n'existe pas.

Exemple =

- Une table > a table
- Un cahier > a notebook

Exemple de verbe conjugué au présent

Speak = Parler

- <span style="color:green">I</span> speak
- <span style="color:green">You</span> speak
- <span style="color:green">He/She/It</span> speak<span style="color:red">s</span>
- <span style="color:green">We</span> speak
- <span style="color:green">They</span> speak

See = Voir

- <span style="color:green">I</span> see
- <span style="color:green">You</span> see
- <span style="color:green">He/She/It</span> see<span style="color:red">s</span>
- <span style="color:green">We</span> see
- <span style="color:green">They</span> see

Particularité pour les verbes qui se terminent par un "**O**", on va ajouter "**ES**".

Il y a 2 cas seulement : 

Do = Faire

- <span style="color:green">I</span> do
- <span style="color:green">You</span> do
- <span style="color:green">He/She/It</span> do<span style="color:red">es</span>
- <span style="color:green">We</span> do

- <span style="color:green">I</span> go

#### 1.1.2. La formation d'une phrase

<span style="color:green">Sujet</span> + <span style="color:lightblue">Auxiliaire</span> + <span style="color:red">Verbe</span>

Auxiliaire "Do", on le traduit pas, il est présent pour faire une phrase

Do - Do not (don't)

Exemple: 

<span style="color:green">J'</span><span style="color:red">aime</span> le chocolat = <span style="color:green">I</span> <span style="color:lightblue">do</span> <span style="color:red">like</span> chocolate

Au présent l'auxiliaire n'est pas obligatoire.

le "do" accentue le verbe.

<span style="color:green">J'</span><span style="color:red">aime</span> le chocolat mais<span style="color:green">j'</span> <span style="color:red">adore</span> le caramel = <span style="color:green">I</span> <span style="color:red">like</span> chocolate but <span style="color:green">I</span> <span style="color:lightblue">do</span> <span style="color:red">love</span> caramel

Exemple avec une phrase à la négation.

- <span style="color:green">Je</span> <span style="color:purple">n'</span><span style="color:red">aime</span> <span style="color:purple">pas</span> le chocolat = <span style="color:green">I</span> <span style="color:lightblue">don't</span> <span style="color:red">like</span> the chocolate
- <span style="color:green">Je</span> <span style="color:red">fais</span> du sport = <span style="color:green">I</span> <span style="color:red">do</span> sport
- <span style="color:green">Je</span> <span style="color:purple">ne</span> <span style="color:red">fais</span> <span style="color:purple">pas</span> de sport = <span style="color:green">I</span> <span style="color:lightblue">don't</span> <span style="color:red">do</span> sport


Exemple:

- <span style="color:green">J'<span><span style="color:red">ai</span> un chien = <span style="color:green">I</span> <span style="color:red">have</span> a dog
- <span style="color:green">Je</span> <span style="color:purple">n'</span><span style="color:red">ai</span> <span style="color:purple">pas</span> de chien = <span style="color:green">I</span> <span style="color:red"><s>haven't</s></span> a dog

> C'est incorrect. **have** est un auxiliaire donc on peut mettre la négation sur **have** mais pour le présent simple, on utilise uniquement l'auxiliaire **do**.

- <span style="color:green">Je</span> <span style="color:purple">n'</span><span style="color:red">ai</span> <span style="color:purple">pas</span> de chien = <span style="color:green">I</span> <span style="color:purple">don't</span> <span style="color:red">have</span> a dog 

Il y quand même une petite exception avec le verbe **Be**. On met la négation sur le verbe.

<span style="color:green">Sujet</span> + <span style="color:lightblue">auxiliaire (Be)</span> + <span style="color:red">verbe</span>

Exemple :

<span style="color:green">Je</span> <span style="color:red">suis</span> une fille = <span style="color:green">I</span> <span style="color:red">am</span> a girl

<span style="color:green">Je</span> <span style="color:purple">ne</span> <span style="color:red">suis</span> <span style="color:purple">pas</span> une fille = <span style="color:green">I</span> <span style="color:red">am not</span> a girl

> Il y uniquement avec le verbe **be** où on aura pas l'utilité de rajouter l'auxiliaire **do**.
>
> Mais sinon avec le verbe **do** et avec le verbe **have**, je suis dans l'obligation d'utiliser l'auxiliaire **do** à la négation pour faire ma phrase, sinon la phrase est incorrecte.

Verbe **Be** à la négation

- I am not
- You are not (aren't)
- He/She/It is not (isn't)
- We are not
- They are not

#### 1.1.2.a. Quand on parle d'une habitude ou d'une routine

Exemple : 

- <span style="color:green">I</span> <span style="color:red">wake up</span> every morning at 7 a.m = Je me lève tout les matins à 7 heures
- <span style="color:green">I</span> <span style="color:lightblue">do</span> <span style="color:red">brush</span> my hair every = Je me brosse mes cheveux tous les jours.

#### 1.1.2.b. Quand on parle d'un fait

Exemple :

- <span style="color:green">I</span> <span style="color:red">have</span> <span style="color:lightblue">a phone</span> = J'ai un téléphone
- <span style="color:green">I</span> <span style="color:lightblue">don't</span> <span style="color:red">live</span> in Paris = Je ne vis pas à Paris

#### Rappel

On utilise le présent simple quand on parle de :

- Une habitude; une routine
- Un fait

#### 1.2.1 Les pronoms/adjectifs possessifs

##### Les adjectifs possessifs

|Sujet|Adjectifs possessifs|Traduction|
|:---:|:------------------:|:--------:|
|I    |My                  |Mon/Mes   |
|You  |Your                |Ton/Tes   |
|He   |His                 |Son/Ses   |
|She  |Her                 |Son/Ses   |
|It   |Its                 |Son/Ses   |
|We   |Our                 |Notre/Nos |
|They |Their               |Leur/Leurs|

##### Les pronoms possessifs

|Sujet|Pronoms possessifs |Traduction|
|:---:|:-----------------:|:--------:|
|I    |Mine               |Mien(s)/Mienne(s)   |
|You  |Yours              |Tien(s)/Tienne(s)   |
|He   |His                |Sien(s)/Sienne(s)   |
|She  |Hers               |Sien(s)/Sienne(s)   |
|It   |Its                |Sien(s)/Sienne(s)   |
|We   |Ours               |Les notres |
|They |Theirs             |Les leurs|


Contrairement à <span style="color:green">l'adjectif possessif</span>, le <span style="color:red">pronom possessif</span> est utilisé seul. Il n'est jamais suivi d'un <span style="color:lightblue">nom</span>

Exemple : 

- Is it your book ? 
  - Yes, it is <span style="color:red">mine</span>
  - Yes, it is <span style="color:green">my</span> <span style="color:lightblue">book</span>
  
- Est-ce ton livre ? 
  - Oui, c'est le <span style="color:red">mien</span>
  - Oui, c'est <span style="color:green">mon</span> <span style="color:lightblue">livre</span>

### 1.2.2 Les pronoms personnels

- Il peut être un sujet

|Francais | Anglais|
|:-------:|:------:|
|Je       | I|
|Tu/Vous  | You|
|<span style="color:red">Il/Elle</span>  | <span style="color:red">He/She/It</span>|
|Nous     | You|
|Vous     | We|
|Ils/Elles| They|

- Il peut être un complément direct ou indirect

|Francais | Anglais|
|:-------:|:------:|
|m', me, moi       |me|
|t', te, toi  |you|
|l', le, lui  |him|
|l', la, lui     |her|
|l', la, le, lui - neutre     |it|
|nous|us |
|les, leur, elles, eux|them|


> Les pronoms, qu'ils soient **sujets** ou **compléments**, s'emploient pour remplacer des noms.

Exemple :

- Tell <span style="color:red">John</span> that you are sorry = Tell <span style="color:red">him</span> that you are sorry
- Dis à <span style="color:red">John</span> que tu es désolé = Dis <span style="color:red">lui</span> que tu es désolé

- I want to help <span style="color:red">Véronica</span> = I want to help <span style="color:red">her</span>
- Je veux aider <span style="color:red">Véronica<span> = Je veux <span style="color:red">l'</span>aider
