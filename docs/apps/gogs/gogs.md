## <i class="fa-solid fa-hashtag"></i> Information

![Logo](../../_media/apps/gogs/gogs_logo.png ':size=250 :no-zoom')


> <i class="fa-solid fa-quote-left"></i> **Gogs** est une solution Git auto-hébergée garantie sans prise de tête et écrite en Go.
>
> Le but de ce projet est de rendre la mise en place d’une solution Git auto-hébergée simple, rapide et sans prise de tête. Avec Go, cela peut être fait avec une distribution binaire indépendante à travers TOUTES les plates-formes qui supportent ce langage, notamment Linux, Mac OS X, Windows, ainsi que ARM. <i class="fa-solid fa-quote-left fa-rotate-180"></i>
